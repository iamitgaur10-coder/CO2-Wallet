# Moss - Luxury Carbon Wallet

The first decentralized wallet for offsetting your real-world lifestyle.

## Tech Stack
- React + Vite
- Tailwind CSS + Framer Motion
- Google Gemini 3 Pro (AI Analysis)
- Spline (3D Visuals)
- Supabase (Backend)
- Toucan Protocol (Carbon Retirement)

## Setup
1. `npm install`
2. `npm run dev`
